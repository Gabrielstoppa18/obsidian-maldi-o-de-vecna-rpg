# Núcleo de Dragão Vermelho

Presente de casamento dado por **[[Satoru Gojo]]** e **[[Ryomen Sukuna (Buggs)]]** a **[[Edward Elric]]**. Usado por Ed para criar o golem **[[Al (Golem)|Al]]** ("One for All").

## Uso pela Facção Puristas
A **[[Facção Puristas]]** recebeu uma arma alimentada por este núcleo e a usou para destruir **[[Aldun]]**.

## Papel no Plano de Vecna
**[[Vecna (Asloth Ferren)|Vecna]]** pretende usar a energia desta arma para enfraquecer/matar **[[Tenebra]]** e se tornar um dos **[[Os Vinte]]**.
