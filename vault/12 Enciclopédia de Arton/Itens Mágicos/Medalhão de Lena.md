# Medalhão de Lena

Acessório maior (T$ 30.000). Quando o usuário é reduzido a 0 ou menos pontos de vida, esta joia emite uma explosão de energia positiva que cura 100 PV antes que ele caia. Este poder só se ativa uma vez por dia.
